Sentencias en Javascript

Punto y coma

Es recomendable usar el ; al final de cada sentencia.
Podemos escribir varias sentencias en la misma linea, separándolas con punto y coma
var a = 1; var b = 2; var c = a+b; alert(c);

Espacios en blanco

Los múltiples espacios en blanco se ignoran
Se recomienda utilizar espacios entre operadores =, +, -, *, /

Líneas de más de 80 caracteres

Es recomendable dividir la línea después de un operador

Bloques de código

Se agrupan entre llaves
Se utilizan en funciones, bucles, condiciones, etc.

  • if...else
